# CA - 5

Dmytrii Puzyr

### Usage

1. `git clone https://github.com/yugisu/ca-5 yugisu-ca-5 && cd yugisu-ca-5`
1. `make` to compile
1. `./build/main <keys>` to run the program

### List of accepted parameters

- `-c (--create)`
- `-a (--add)`
- `-f (--file)`
- `-l (--list)`
- `-h (--help)`
- `-v (--version)`
